# Preguntas frecuentes para otras soluciones de CX Enterprise{#faqs-for-other-experience-cloud-solutions}

Las preguntas más frecuentes sobre las características, la funcionalidad y los problemas relativos al uso de otras soluciones de CX Enterprise con el servicio de ID de visitante.

## Analytics y Audience Manager {#section-b3dd206d497041acb04554c6fb1c912a}

**¿El historial de visitas de un usuario se exportará de Adobe Analytics a Audience Manager después de implementar el servicio de ID de visitante?**

Aquí existen dos opciones:

* Si un usuario realiza cualquier actividad de visita una vez implementado el servicio de ID de visitante, el visitante y su historial se incluyen en la exportación de datos a Audience Manager.
* Si un usuario no realiza ninguna actividad de visita una vez implementado el servicio de ID de visitante, el visitante y su historial no se incluirán en la exportación de datos a Audience Manager. Como no hay nueva actividad, no hay forma de asociar el ID de Analytics con el ECID.
