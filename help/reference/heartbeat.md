# Video Heartbeat y el servicio de ID de visitante de Adobe {#video-heartbeat-and-the-experience-cloud-id-service}

Video Heartbeat requiere el servicio de ID de visitante. Cuando se mide vídeo en JavaScript o Flash, la implementación del servicio de ID de visitante en las páginas del sitio y en la reproducción del vídeo garantiza que los visitantes no se contabilicen por separado cada vez que reproduzcan el vídeo.
